# Новый логотип для сайта Липецк Поверка

## 📋 Описание

Создан современный текстовый логотип, который заменяет изображение SVG.

## 🎨 Дизайн логотипа:

### Структура:
```
ЛИПЕЦК
Поверка счетчиков
```

### Характеристики:
- **Основной текст "ЛИПЕЦК":**
  - Цвет: Синий (#007bff) - фирменный цвет Bootstrap
  - Размер: 28px (десктоп), адаптируется на мобильных
  - Начертание: Жирный (700)
  - Эффект: Лёгкая тень для объёма
  - Интервал между буквами: 1px

- **Подзаголовок "Поверка счетчиков":**
  - Цвет: Серый (#6c757d)
  - Размер: 13px
  - Стиль: Заглавные буквы (uppercase)
  - Интервал между буквами: 2px

### Интерактивность:
- ✅ Плавное увеличение при наведении (scale 1.05)
- ✅ Анимация перехода 0.3 секунды
- ✅ Полностью адаптивный дизайн

## 📱 Адаптивность:

- **Десктоп (>768px):** 28px / 13px
- **Планшет (768px):** 24px / 11px
- **Мобильный (<480px):** 20px / 10px

## 📁 Файлы:

1. **index.html** - обновлённая разметка логотипа
2. **css/logo-custom.css** - стили логотипа

## ✨ Преимущества:

✅ Не требует загрузки изображений  
✅ Быстрая загрузка страницы  
✅ SEO-дружественный текст  
✅ Легко редактируется  
✅ Полностью адаптивный  
✅ Современный дизайн  
✅ Уникальный для Липецка  

## 🔄 Где используется:

1. Шапка сайта (navbar)
2. Футер сайта
