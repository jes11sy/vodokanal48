# Водоканал48 — поверка счетчиков воды в Липецке

Сайт [водоканал48.рф](https://водоканал48.рф) — поверка, установка и замена счетчиков воды на дому.

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Главная страница |
| `thanks.html` | Страница после отправки заявки |
| `site-config.js` | Публичные настройки сайта (домен, контакты, метрика) |
| `config.js` | Секреты Telegram-бота (не в git) |
| `config.example.js` | Шаблон для `config.js` |
| `script.js` | Отправка заявок в Telegram |
| `yandex-metrika.js` | Яндекс.Метрика |
| `robots.txt` / `sitemap.xml` | SEO |

## Установка

1. Скопировать `config.example.js` → `config.js`
2. Вписать `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`
3. Залить файлы на хостинг домена `водоканал48.рф`
